 Servidor de Smartlocate

> *Transforming communication in confined spaces*

---

## Resumen
El servidor de Smartlocate (VSDR-TG), desarrollado por UQOMM, proporciona una solución de visualización en tiempo real de la ubicación del personal dentro de minas subterráneas.
* Ofrece una visión actual y precisa de la situación a través de mapas 2D.
* Permite monitorear la ubicación de los trabajadores en todo momento, mejorando la eficiencia operativa.
* Cuenta con un módem de radio integrado diseñado para operar en ambientes con alto ruido, lo que permite recuperar datos de cada unidad interrogada.
* Está basado en un sistema operativo Linux con un servidor web embebido para acceso remoto al agregar el dispositivo a la Red de Área Local (LAN).

## Descripción Detallada
El Servidor de Smartlocate de UQOMM (VSDR-TG) para la red Leaky Feeder permite el sondeo bidireccional de datos.

* **Capacidad:** Interroga hasta 50 TAGS por cada Smartlocate Sniffer para asegurar la conexión y el estado de cada dispositivo.
* **Conectividad:** El cable Leaky Feeder funciona como vía de transmisión de datos para el servidor. Es una solución no invasiva y plug-and-play.
* **Visualización:** Proporciona una visualización dinámica y actualizada de la ubicación de cada personal, además de una identificación y conteo rápido de personas en zonas específicas.
* **Interfaz de Usuario:** El VSDR-TG utiliza su mini-PC integrada para ofrecer acceso vía navegador al software dedicado. Este software puede ser accesado localmente o de forma remota conectando el servidor a la LAN del sitio.
* **Datos:** Incluye información de cada TAG, como el nombre asociado, ID, estado, porcentaje y voltaje de la batería. El software presenta esta información mediante indicadores de alarma.

## Instalación y Mantenimiento

### Pasos de Instalación
1.  **Montaje y Energía:** Instale el servidor en un rack de 19 pulgadas (2U de altura), conecte el cable RF al HeadEnd y suministre alimentación de 12 VDC.
2.  **Encendido:** Encienda la unidad presionando el botón PWR; se encenderá.
3.  **Red:** Conecte el servidor a la LAN mediante el puerto ETH; se encenderá cuando el servidor esté activado.

### Operación y Mantenimiento
* **Monitoreo:** Verifique el estado de conexión del Smartlocate Sniffer diariamente y supervise los valores del sniffer semanalmente.
* **Gestión:** Agregue, mueva o elimine sniffers para que coincidan con la red real en el software, con comandos fáciles disponibles para esto.
* **Configuración:** Las frecuencias pueden cambiarse usando un software especial, disponible bajo petición.
